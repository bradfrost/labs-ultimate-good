# Save the World

*The world's biggest problems. What actually works.*

A serious, hopeful, evidence-based reference site summarizing humanity's biggest problems and the most pragmatic solutions to each. Built with the [Eddie Design System](https://bradfrost.com).

## The Point

This site is one half of a pair. Its sibling — [End the World](../ultimate-bad/) — uses the **exact same design system**, same components, same layout patterns. The only difference is one accent color token and the content.

The tools are neutral. The maker is not.

## Running Locally

```bash
npm install
npm run dev
```

## Building

```bash
npm run build
npm run preview
```

## Tech Stack

- Vanilla HTML + Eddie Design System web components
- Vite for dev server and production build
- No framework — static HTML pages
- Eddie Design Tokens (BFW theme) with teal accent override
